Space Invaders Game

This was my final project in the TTA Programme. I wanted to produce an Object Oriented version of Space Invaders however, I initially struggled to build this in JavaScript. Utilising my growth mindset I asked my teacher if there was another language better equipped for my project. He suggested I use Typescript, which I adapted to within the week. I had fun developing this project as it was interesting to make every element into an object (from the invaders to the bullets!). This allowed for characteristics such as “alive” or “dead”.

If I were to better this project, I would add additional features such as a “score counter”/“Game Over” screen, and the ability for the invaders to shoot back.

(Controls: left and right arrows for movement, Spacebar to shoot)

Horse Racing Gaming

I utilised a mix of JavaScript, HTML and CSS to create a horse racing game which generates a random winner each time. This is where I learnt the importance of arrays and math.random and how easily they can simulate a basic horse race.

Splitter

This was another project on the TTA course where I merged CSS with JavaScript to make an entertaining webpage. When you click on the page, it splits it into four random colours. Clicking again will repeat this, but localised in that quarter section. You can continue doing this infinitely (and remove squares with shift-click) making for some interesting patterns. This was made possible with EventListeners.

Lazatus -Choose your own adventure Game

This was my passion project in Python. Lazatus is a Lord of the Rings inspired homage to the classic text-based choose-your-own-adventure games of the 80s. I added my own twist where you can choose a race which gives you buffs and perks. I hope to add three different endings based on the players' decisions. I will add this to my GitHub soon.
